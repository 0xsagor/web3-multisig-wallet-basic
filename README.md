# Web3 MultiSig Wallet Basic

This project helps beginners learn how multi-signature wallets work on blockchain.

You will learn:
- What a multi-signature wallet is
- How to connect a wallet
- How to create a transaction
- How to approve a transaction
- How to execute a transaction

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-multisig-wallet-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- multisig.js = MultiSig contract functions  
- web3.js = Blockchain connection  
- deploy.js = Contract deploy script  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Create transaction  
- Approve transaction  
- Execute transaction  
- Show transaction status  

## Learning Goal
This project explains how shared control wallets work using blockchain.
